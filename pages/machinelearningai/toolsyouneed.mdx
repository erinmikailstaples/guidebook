# Tools of the Trade

## Common Tools used in Machine Learning 

### Languages and Programs

- [Python](https://packaging.python.org/en/latest/tutorials/installing-packages/) the latest version is Python 3.  Follow the above link to install Python.
- [pip](https://pypi.org/project/pip/)is the package install er for Python. You can use it to download various packages and tools from the [Python Package Index](https://pypi.org/project/pip/)
- [Anaconda](https://www.anaconda.com/) is a data science platform that allows you to perform machine learning on a single machine. The tool also has Anaconda Notebooks, which is a cloud-hosted notebook, allowing you to learn right from the browser. 
- [Jupyter Notebooks/Jupyter Lab](http://jupyter.org/)free, and open software for creating development notebooks.
